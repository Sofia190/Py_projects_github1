Animal Atlas

Animal Atlas is an online collection of animal species which offers helpful information on this subject. Because its database is very large, some functionality should help the users confortably navigate through the web page. 


Here is the general structure that Animal Atlas uses to store data:


Dict({'animal': "def", 'mammal'/'other': "Yes"/"category", 'URL':"url"(picture)})


- retrieve animal definitions by key

- retrieve all names starting with letter...

- retrieve animal names which belong to mammal or other

- check if an animal belongs to mammal or other category

- view picture for each animal
